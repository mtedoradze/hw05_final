Вот какие инструменты были ипользованы для создания этого сайта:

  • С помощью Django ORM выполняется работа с базой данных, в которой хранятся посты пользователей, информация о самих пользователях и другая техническая информация.
  • Для оформления страниц в шаблонах использован фреймворк Twitter Bootstrap
  • Для управления доступом пользователей применен декоратор Django @login_required
  • Также применены дополнительные утилиты, такие как контекст-процессоры (для определения в шаблоне авторизован пользователь или нет, для добавления универсальных переменных в шаблоны) и паджинатор
  • Для добавления картинок к постам использовано приложение sorl-thumbnail
  • и много других полезных библиотек и приложений
